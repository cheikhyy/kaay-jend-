# kaay-jend-
Créer une plateforme permettant de centraliser la vente en ligne de meubles d'occasion (sur le modèle de Vinted).  L'objectif est de permettre aux étudiants étrangers de vendre sereinement leurs meubles à la fin de leur cursus scolaire afin de faciliter leur retour dans leur pays d'origine, tout en évitant le gaspillage.
